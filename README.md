# SaveTest TMS Project

Это проект SaveTest TMS для управления тест-кейсами.

## Структура проекта

- `tests/cases/` - тест-сьюты
- `tests/attach/` - вложения
- `tests/common-case/` - общие кейсы
- `.save-test` - конфигурация проекта

## Использование

1. Создавайте тест-сьюты в папке `tests/cases/`
2. Используйте контекстное меню для управления файлами
3. Применяйте горячие клавиши для быстрой работы
